# CampusOps
A lightweight operations management platform for student organizations and small teams. Built to streamline event planning, driver coordination, role-based access, and internal logistics in one centralized dashboard.


# For Codepath Week 2 requirements (Goals and Required Features):

✅ Create a PostgreSQL database and table
✅ Display data from a PostgreSQL database in a web app
✅ The web app uses only HTML, CSS, and JavaScript without a frontend framework.
✅ The web app is connected to a PostgreSQL database, with an appropriately structured database table for the list items.
